pub enum InferenceValue<I: Interner> {
    Unbound(UniverseIndex),
    Bound(GenericArg<I>),
}

The value of an inference variable. We start out as Unbound with a universe index; when the inference variable is assigned a value, it becomes bound and records that value. See EnaVariable for more details.
